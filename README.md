# rec-movielens

![License](https://img.shields.io/badge/License-MIT-blue)

---

### Introduction

A simple implementation of Recommender System based on the [MovieLens 1M Dataset](https://grouplens.org/datasets/movielens/1M/).

---

### Quick Start

#### Training

```bash
python train.py
```

#### Evaluation

```bash
python evaluate.py
```
